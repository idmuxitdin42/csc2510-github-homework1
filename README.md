# CSC 2510 Homework 2

Howdy everyone! 

## Goal

The goal of this homework is to build muscle memory in regards to creating branches and merging. During class, you all voted for the more... chaotic approach of everyone working off the same branch. Surely nothing bad will happen. 

## Rules

The usual - nothing illegal and nothing I have to fill out paper work for. 

## Homework Steps

From the most recent commit on the main branch on the repository, you will create a new branch. I'm deciding on Python for kicks and giggles. 

You'll need to create two functions - one will be named *func{LastName}*. Mine will be **funcVandergriff**. You can do whatever your heart desires in this function, as long as it does not effect your classmates (no malware, no 100 million lines, etc). 

The second one will be your true merge practice - the other required function will be called **fightClub**. You will have to decide to either keep your code, or the code you are trying to merge into - or some hybrid! 

Finally, add your personal function call to the main section. 

1. Branch off of the most recent version of main
2. Create funcLastName
3. Create fightClub
4. MERGE!!!
